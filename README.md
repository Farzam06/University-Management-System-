# University-Management-System
A University Networking Management System designed using Cisco Packet Tracer that simulates a real-world campus network with multiple departments. The project implements VLANs, routing protocols, servers, and security features to ensure efficient, secure, and scalable communication across the network.

Overview
This project is a University Networking Management System designed and simulated using Cisco Packet Tracer. It demonstrates how a real-world campus network can be structured to ensure efficient communication, proper resource sharing, and secure data transmission across multiple departments.

Objectives
Design a scalable and organized university network
Enable communication between different departments
Implement network security and segmentation
Simulate real-world networking concepts
Network Structure

The network is divided into multiple departments, including:

Administration
Faculty
Students
Library
IT Department

Each department is connected through switches, while routers are used to interconnect different networks and enable routing.

Key Features

IP Addressing: Static and/or DHCP configuration for all devices
VLANs: Logical segmentation of departments for better management and security
Routing Protocols: Implementation of protocols such as RIP/OSPF
Servers: Integration of services like DNS, HTTP, FTP, and Email
Access Control Lists (ACLs): Restrict unauthorized access between networks
Inter-VLAN Routing: Communication between different VLANs
Technologies Used
Cisco Packet Tracer
Networking Devices (Routers, Switches)
Networking Protocols (IP, VLAN, Routing Protocols)

Project Functionality
Devices within the same department can communicate locally
Inter-department communication is enabled through routing
Servers provide centralized services to all users
Security policies control access between departments

Learning Outcomes
Understanding of network design and topology
Hands-on experience with VLANs and routing
Knowledge of network security implementation
Practical exposure to real-world networking scenarios

How to Run
Open the .pkt file using Cisco Packet Tracer
Explore the network topology
Test connectivity using ping and other simulation tools
Review configurations of routers, switches, and servers
